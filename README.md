  <h2>Simple Snapchat like app</h2>
</p>
<br>

  ### :zap: Features:
  - [x] Create User account
  - [x] Send snaps(Choose from Device storage)
  - [x] Send small message along with snap
  - [x] View Snap
  - [x] Automatically delete snap after view
